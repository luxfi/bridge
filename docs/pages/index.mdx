## What is Lux Bridge?

The Lux Bridge is a decentralized cross-chain bridge that enables:

- **Secure Asset Transfers** - Transfer tokens between 15+ blockchain networks
- **Multi-Party Computation** - 2-of-3 threshold signatures for maximum security
- **Universal Message Passing** - Send arbitrary messages across chains
- **Low Fees** - Competitive 0.3% bridge fee with caps
- **Fast Finality** - Transfers complete in minutes, not hours

## Getting Started

```bash
# Clone the repository
git clone https://github.com/luxfi/bridge
cd bridge

# Install dependencies
pnpm install

# Start local development
make up
make start-mpc-nodes
cd app/bridge && pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the bridge UI.
